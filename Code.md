# Source code related to Logan

* The **cloud infrastructure used to construct Logan** is solely intended for internal usage due to its high execution costs, its source code is publicly available at https://gitlab.pasteur.fr/rchikhi_pasteur/erc-unitigs-prod/.
* The pipeline and infrastructure source code of **Logan-DIAMOND and Logan-minimap2**, for shallow and deep homology search, is available at https://gitlab.pasteur.fr/rchikhi_pasteur/logan-analysis.
* The code used to identify **plasmids** via detecting circular contigs is https://gitlab.pasteur.fr/rchikhi_pasteur/logan-circles
